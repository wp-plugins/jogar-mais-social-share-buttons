=== Jogar Mais - Social Share Buttons === Contributors: Victor Freitas Donate link: https://www.paypal.com/cgi-bin/webscr?cmd=_s-xclick&hosted_button_id=X7BF5KKYQMA8E Tags: share, social, buttons Requires at least: 3.0.0 Tested up to: 4.2.2 Stable tag: 0.1 License: GPLv2 or later License URI: http://www.gnu.org/licenses/gpl-2.0.html Este plugin implementa os botões de compartilhamento das princípais redes sociais, inclusive o Whats App. == Description == Insere botões de compartilhamento das redes sociais. Os botões podem ser inseridos via shortcode ou método php. 1. Acrescentará antes, depois do conteúdo dos posts ou em ambos: "O conteúdo deste comentário é de responsabilidade de: {AUTOR}" 2. Armazenará algumas options com os dados que serão usados para mostrar, ocultar, estilizar e remover botões específicos. * As redes suporta são: - Facebook; - Google Plus; - Twitter; - Linkedin; - Pinterest; - Thumbler; - Whatsapp; - SMS; - Gmail; - Mailto; - Print; == Installation == 1. Faça upload dos arquivos para a pasta `/wp-content/plugins/` (mantenha a pasta original do plugin) 1. Ative o plugin na interface de 'Plugins' do WordPress 1. Clique em configurações para ir até a página do plugin ou pelo menu Social Buttons == Frequently Asked Questions == Algumas redes sociais já temos contadores de compartilhamentos.
